## Implementada utilizado: ##

- AngularJS
  - RequireJS
  - Grunt
  - Less
- Testes Unitários:
  - Karma
  - Jasmine

## Para rodar o projeto: ##

```bash
npm install bower -g
npm install
bower install
npm run serve
```
